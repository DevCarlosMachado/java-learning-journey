# Java Learning Journey

Repositório criado para documentar minha evolução nos estudos da linguagem Java.

Aqui registro conceitos aprendidos, exercícios, desafios e projetos desenvolvidos durante minha jornada de aprendizado, desde os fundamentos da linguagem até aplicações utilizando tecnologias do ecossistema Java.

---

## 📚 Conteúdos estudados

### 🔹 Fundamentos da linguagem

* Sintaxe da linguagem Java
* Estrutura sequencial
* Estruturas condicionais
* Estruturas repetitivas

### 🔹 Programação Orientada a Objetos (POO)

* Classes e objetos
* Construtores
* Palavra reservada `this`
* Sobrecarga
* Encapsulamento
* Herança
* Polimorfismo
* Interfaces

### 🔹 Estruturas de dados

* Comportamento de memória
* Arrays
* Listas
* Generics
* Set
* Map

### 🔹 Recursos da linguagem Java

* Manipulação de data e hora
* Enumerações
* Composição

### 🔹 Tratamento de erros e arquivos

* Tratamento de exceções
* Leitura e escrita de arquivos
* FileReader
* BufferedReader

### 🔹 Programação funcional

* Expressões Lambda
* Streams

### 🔹 Banco de dados

* JDBC
* Mapeamento objeto-relacional (ORM)
* JPA/Hibernate

### 🔹 Desenvolvimento Web

* Spring Boot
* Web Services
* Integração com bancos relacionais
* Banco de dados NoSQL com MongoDB

### 🔹 Interface gráfica

* JavaFX
